Falsy | Truthy

No more BS.

Falsy | Truthy is a crowdsource verification platform.


There are times when things get passed on the internet(event/news/social), it sometimes may come off as real but may contain missing information, such as before the video recording or trimmed off videos, misinterpretation of a speech, and many more.

Falsy | Truthy is designed for users to vote their opinions on the article.

How does it work?
If a user submits a source:

  • the user must vote by typing a ^
  • the user must submit an appropriate link
  • user's vote remains anonymous
If a user replies to a comment:
  • No need to vote but can choose to
If a user VOTES to comment:
  • Whatever the original user voted for, the current user will attribute to half a point of agreement/disagreement.
  • Example: If the original user (A) voted ^Real and posted a url with a comment. (B) user reads (A)'s comments and agrees, then selects "Truthy". This will add to the original ^Real vote.
    If user (B) doesn't like (A)'s comments then ^Real points is deducted and ^Neutral points is added.
    Comment vote points are only 1% of the total.
If a user replies to the article:
  • a vote is needed

Vote counts are not disclosed until 100 votes achieve.
